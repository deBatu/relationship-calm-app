# Pundik 🤍

Ruhige Mobile-first Web-App/PWA für zwei Personen. Die App ist absichtlich kein Chat:
eine Aussage pro Zeile, strukturiert in 6 Bereiche, plus Abschlussansicht.

## Bereiche (fix)

1. Was ich an dir mag / schätze
2. Was ich falsch gemacht habe
3. Was ich mir von dir wünsche
4. Was ich selbst besser hätte machen können
5. Was wir künftig besser machen wollen
6. Meine schönste Erinnerung mit dir (max. 3 Zeilen pro Person)
7. Abschlussansicht (inkl. gemeinsames 24h-Ziel + Danke-Satz)

## Verhalten

- Pro Eingabe nur eine Zeile
- Nach Absenden: Feld wird geleert, Zeile wird darunter angezeigt
- Antworten der anderen Person erscheinen nach manuellem Neuladen
- Zugriff über gemeinsamen 6-stelligen Code
- Kein Login
- Am Ende: gemeinsames Mini-Ziel für 24h und je ein Danke-Satz

## 1) Supabase konfigurieren

In `config.js` eintragen:

```js
window.APP_CONFIG = {
  SUPABASE_URL: "https://YOUR-PROJECT.supabase.co",
  SUPABASE_ANON_KEY: "YOUR_SUPABASE_ANON_KEY",
};
```

## 2) Supabase SQL ausführen

Im Supabase SQL Editor:

```sql
create table if not exists section_lines (
  id bigint generated by default as identity primary key,
  relationship_code text not null check (relationship_code ~ '^[0-9]{6}$'),
  author_id text not null,
  section_key text not null check (
    section_key in (
      'appreciation',
      'my_mistakes',
      'wishes',
      'my_self_reflection',
      'future_rules',
      'best_memory'
    )
  ),
  line_text text not null check (char_length(trim(line_text)) > 0),
  created_at timestamptz not null default now()
);

create index if not exists idx_section_lines_session
on section_lines (relationship_code, section_key, created_at);

alter table section_lines
drop constraint if exists section_lines_section_key_check;

    'my_mistakes',
    'wishes',
    'my_self_reflection',

    'best_memory'

alter table section_lines enable row level security;

drop policy if exists "anon_select_section_lines" on section_lines;

on section_lines
for select
to anon

to anon

create table if not exists shared_24h_goals (
```

## 3) Lokal starten

Projektordner öffnen und starten:

```bash
npx serve
```

Danach im Browser die lokale URL öffnen (z. B. `http://localhost:3000`).

## 4) Gemeinsam nutzen

1. Beide öffnen die gleiche URL
2. Beide geben denselben 6-stelligen Code ein
3. Jede Aussage als einzelne Zeile absenden
4. Für neue Einträge der anderen Person auf **Neu laden** klicken
5. Am Ende **Abschlussansicht laden** für die Gesamtansicht
